My version of the late 90's Tamagachi and my second project from after two weeks of learning Java and being introdced to unit testing. Here we have Hold the dog with some simple stats of hunger, water, waste, bored amd tiredness. However, the stats will affect one another. For example, when you feed Hold the water drops. virtual-pets-amok

After 4 weeks of learning Java, this project combined all that was learned within a 4 week time frame. Taking virtual-pet from week 2 and recreated it using not just a single pet but a multitude of pets like Dogs, Cats and Robot Dogs inside a Shelter. User has to feed and water the Dogs and Cats, as well as waste clean up created by each dog when prompted to. The Cats, there's a Shelter litter box which has a counter that resets to 0 when the User decides to clean the litter box. Other features include the intake and outake of pets. Best of all, the User has the ability to see a list of each pet and their stats at all times.

Created using: Java.OOP.TDD.Git/ Eclipse IDE
